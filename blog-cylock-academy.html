    CYLOCK: Balancing Realistic Vulnerabilities with Safe Learning

    CYLOCK was built as a complete web security e-learning platform. It includes authentication, course management, and containerized lab workflow while keeping operational safety as a first-class requirement.

    Core Modules

    • User identity and role mapping (student, mentor, admin).
    • Lab provisioning with isolated Docker containers.
    • Secure REST APIs with JWT and role-based access control.
    • Progress tracking and challenge scoring.

    Security Design Decisions

    @PreAuthorize("hasAnyRole('ADMIN','MENTOR')")
@PostMapping("/labs/{id}/reset")
public ResponseEntity<?> resetLab(@PathVariable Long id) {
    labService.reset(id);
    return ResponseEntity.ok().build();
}

    Important point: vulnerable lab code is intentionally weak for learning, but control-plane services remain hardened and monitored.

    Multi-type Image Blocks

    Threat modeling should include both student attack path and admin control path. Educational platforms still need production-grade access control.
