    Automated SQLi Scanner for Fast Vulnerability Triage

    This project was implemented to quickly detect Error-based, Boolean-based, and Time-based SQL Injection in web applications. Main objective: minimize manual trial-and-error and provide reproducible scan output for security review.

    Architecture and Workflow

    Core pipeline:

    1. Target probing and endpoint discovery.
    2. Form/parameter extraction.
    3. Payload injection and baseline comparison.
    4. Response classification and severity tagging.
    5. Structured HTML report generation.
    # pseudo pipeline
for endpoint in targets:
    params = enumerate_parameters(endpoint)
    for param in params:
        payloads = get_payloads(param)
        for payload in payloads:
            result = send_probe(endpoint, param, payload)
            classify_and_store(result)
